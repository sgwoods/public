Masters of Mathematics Renovation: AbTweak

This project is the starting point for reviving the code and research artifacts associated with Steven Woods' Masters of Mathematics thesis work on AbTweak, a hierarchical nonlinear planner developed at the University of Waterloo.

The repository has moved beyond the initial bootstrap stage and is now focused on validating a runnable SBCL baseline, comparing restored planning modes, and documenting where the recovered code aligns with or diverges from the thesis and papers.

Public project note: active restoration work is underway. As of March 16, 2026, the repository reports a working SBCL baseline across multiple historical planning benchmarks, with ongoing comparison work around Hanoi-4 hierarchy and frontier behavior.

Current State

  • The working copy solves `blocks` / `sussman` in both `tweak` and `abtweak`.
  • The restored baseline solves `registers` in `tweak` and runs `hanoi-3` plus `macro-hanoi` in both modes.
  • `simple-robot-2` shows the expected left-wedge effect in `abtweak`.
  • `hanoi-4` now runs to a bounded search outcome under the restored BFS path, with ongoing comparison work on hierarchy and frontier choices.
  • The repository includes historical snapshots, a current SBCL working tree, and focused documentation for baseline selection, algorithm correspondence, and benchmark interpretation.

Renovation Priorities

  1. Keep tightening the runnable SBCL baseline against the historical papers, thesis, and recovered snapshots.
  2. Document how benchmark behavior changes across hierarchy and control choices, especially on larger Hanoi cases.
  3. Preserve clear provenance between archival code, working copies, and interpretive modernization steps.
  4. Use the repository documentation as the public working log for restoration decisions and benchmark evidence.
