Masters of Mathematics Renovation: AbTweak

This project is the starting point for reviving the code and research artifacts associated with Steven Woods' Masters of Mathematics thesis work on AbTweak, a hierarchical nonlinear planner developed at the University of Waterloo.

The immediate goal is to recover the original algorithmic context, document the historical implementation, and prepare a clean foundation for reconstructing and evaluating the planner in a modern repository.

Public project note: the project documentation bootstrap is underway. The initial emphasis is on source provenance, references, and implementation recovery rather than modernization.

Core References

  • Josh D. Tenenberg and Qiang Yang, "ABTWEAK: Abstracting a Nonlinear, Least Commitment Planner", AAAI-90.
  • Qiang Yang, Josh D. Tenenberg, and Steve Woods, "Abstraction in Nonlinear Planning", University of Waterloo Technical Report CS-91-65, December 1991.
  • Steven G. Woods, "An Implementation and Evaluation of a Hierarchical Nonlinear Planner", University of Waterloo Technical Report CS-91-17, March 1991.
  • CMU AI Repository package page for `abtweak`.

Renovation Priorities

  1. Recover the historical terminology, claims, and evaluation criteria from the papers and thesis.
  2. Locate or reconstruct the original implementation artifacts and example planning domains.
  3. Document any differences between the archived code, the thesis, and the published research.
  4. Prepare a reproducible baseline before attempting broader modernization work.
