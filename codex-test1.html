Current state

Primary goal

Ship a tight, polished four-stage Galaga tribute: Stage 1, Stage 2, Stage 3 challenging stage, and a fair Stage 4 finish.

What is already solid

The game is live in the browser with build identity, high-score persistence, replay harnesses, challenge-stage structure, capture/rescue flow, and synced public release metadata.

Current blocker

Stage 4 fairness is the last stubborn gameplay knot. The earlier stages are now in much better shape and the release dashboard tracks the remaining path to 1.0.

Latest release note

Capture clarity, boss feedback, and 1.0 release focus

Improved rescue-versus-destroy feedback, strengthened ship-hit and boss first-hit cues, aligned the carried-fighter attack position with the original game, and promoted visible special squadron bonuses into the four-stage 1.0 release plan.
