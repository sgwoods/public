Codex-Test1: Neo Galaga Tribute

This project is a Galaga tribute game built from the bottom up as a web application, with active work on arcade feel, stage flow, capture-and-rescue mechanics, and browser-based playability.

The repository combines live gameplay tuning with deployment, feedback collection, and local replay tooling so the game can be iterated quickly while staying playable as a public browser experience.

Public project note: active game development is underway. As of March 19, 2026, the latest visible release work is Build identity, settings drawer, and roadmap sync.

Current State

  • The game runs as a browser-based arcade shooter with keyboard controls and integer-scaled playfield presentation.
  • Implemented features include stage progression, challenge stages, boss capture beam behavior, ship rescue, dual-fighter mode, and tuned dive/missile pacing.
  • The project includes built-in gameplay logging plus local replay and batch-analysis harnesses for balancing and regression checks.
  • The public build supports in-game feedback submission and exported session logs to aid ongoing tuning.
