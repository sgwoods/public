Caeti Demo

University of Hawaii

Overview

This is a tool for developing Army tank platoon exercises. You can specify as little as a few drills to be included in an exercise, and our system will do the work of finding paths through terrain, placing enemy units, and writing OPORDs and exercise tables for a complete exercise that conforms to your requirements. You can also specify particular subregions of terrain and particular assembly areas to be used.

Once the system has generated a scenario from your requirements, you can tweak the scenario by altering your requirements.

Try it

There are two ways of using this demo: you can run the system using your own set of requirements, or you can use a set of requirements that we have prepared (using this set can be helpful in learning how to use the interface).

If you would like more information about this project, please contact Alex Quilici. The project is supported by a DARPA grant.
