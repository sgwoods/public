First read the following paragraph and both questions.  If you {\bf cannot} do
question 1, do question 2.  {\bf Do only one question}.  Note though that
Question 1 is worth 15 marks, while Question 2 is worth only 10 marks.

You are given an array defined as ``int letters[SIZE]'' which has been
filled with values obtained using getchar().  You know that the final or
terminating value in this array is the marker -1. 

\begin{enumerate}
\item Assume you want to sort the array (into alphabetical order, lowercase
before upper), given that 'a' $<$ 'A', and 'a' $<$ 'b'.   Write a {\bf
function} ``void sort\_array( int x[] )'' which sorts the array passed as the
parameter.  The array should be of {\em any size}. 

\item  Assume you want to find the ``smallest'' character (in alphabetical)
order which is represented in the array, given that 'a' $<$ 'A', and 'a' $<$
'b'.  Write a {\bf function} ``int find\_smallest( int x[] )'' which returns
the desired element of the array for an array of {\em any size}.
\end{enumerate}

Hint: Think a moment about what getchar() does before starting.
