Basic UNIX Commands

There is a small set of UNIX commands you need to be familar with in order to use Wiliki. Here are the commands and what they do:
  • ls - List the names of your files.

  • ll - List the names of your files, along with other information such as their size and when you last changed them.

  • more - Display the contents of a file one page at a time.

  • cp file1 file2 - Copy the contents of file1 into file2, erasing the old contents of file2.

  • mv file1 file2 - Rename file1's name to file2, destroying file2 if it already exists.

  • rm file - Delete file (warning: once you do this you can't get the file back, ever).

  • man command - Provide help with command.

  • man -k thing - List all commands involving thing.

  • logout - Exit UNIX system.

  • passwd - Change your password (note: your new password must be at least 5 characters long and contain at least 1 number.

  • cc -Aa file.c - Compile file.c (which should be a C source program and must have a name that ends in .c) using the HP standard ANSI-C compiler.

  • vi file - Start up the visual editor on file (which does not have to exist already).

  • elm - Read your mail. After it starts, hitting the space bar displays the current message, and typing ?? lists all "elm" commands and what they do.

  • elm accountname - Send mail to accountname. It will prompt for a message subject (enter a very brief description of the message, such as "Help on Program 1"), and then place you into the editor. For example, to send mail to the professor, use the command "elm alex".

For all of these commands, file, file1, and file2, stand for any file name, command stands for any UNIX command, and accountname stands for someone's computer account. Note that commands are case-sensitive, so command names must typed in lowercase. You must separate the command and its arguments by spaces.
