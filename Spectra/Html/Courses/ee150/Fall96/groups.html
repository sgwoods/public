Current Announcements:

  • December 4: For the final part of assignment 6 grading is as follows: 50 correctness and code style, 25 write up, 25 demo. Write up should include a better version of assign 5 handin including algorithm descriptions, code description (ie layout, etc, and could include charts, figures and annotations), AND a users manuall which is designed for those who want to use your program to play the game - it should describe all functionality and how to make it all work. AS WELL, each group member must handin one page which states: a) how much as a percentage of the total work did YOU do personally (your estimate), b) how much as a percentage of the total work did each of the other group members do, and c) write a comment as required to explain why you feel as you do.
  • Nov 21: NOTE!!! Assign 5/6 is a GROUP PROJECT. This means that there is only one handin pergroup expected. This applies to both code and reports.
  • Nov 18: Assign 5 Due dates and times are as follows (there is an inconsistency in the handout): The hard-copy report and documentation (which must be typeset/word processed) is due IN-CLASS (2.30pm) Friday, Nov. 22. The code required is due no later than 6pm Friday Nov. 22. Code should be submitted by ONE team member only (include a comment header in each submitted file indicating the group).
  • See also the Project-related Questions and Answers. The project itself is available too.
  • The project will involve: (1) handing in a completed program, (2) handing in a completed set of documentation for your program (to be described on the handout), and (3) performing a demonstration of your program working. The demo will involve the TAs/Prof testing your program and asking questions to team members during the demo. The grade will be loosely divided as 40% program as per usual, 40% demonstration, and 20% documentation.
  • Assignment 5 and 6 are group projects. There will be one shared grade. However, based on post-project evaluations by group members, it may be necessary to adjust the group grade for individuals. Do your best to work together in harmony, however, if there are difficulties please contact me directly BEFORE the problem becomes unmanageable.
