PROBLEM #1 (45 points, 15 points each)

Once again, show what each of the following program fragments do. To determine the answer, work through the code by hand and show us the output that results. You must show how these programs execute by using the ``table'' model we have seen repeatedly in class in order to get full credit.

Hint: The table method of showing the output involves showing the time or execution steps on one axis (say the horizontal axis) and each variable that changes on the other axis (say the vertical axis). You should show the output somewhere in the same table.

Part A

main()
{
  int t, v = 0;
  for (v = 10; t < 6, t > 5; v++)
    {
      printf(" In Loop: t= %i v= %i\n", t, v);  
      t += 2;
      v -= 3;
    }
  printf("Done: v = %i, t = %i\n", v, t);
  return 0;
}

The solution.

Part B

main()
{
  int f1 (int);
  int j = 12;
  int y = 15;
  do
    {
      int y = 10;
      y = f1 ( y );
      printf("In Loop: y = %i, j = %i \n", y, j), --j;
    }
  while( y > 0 && j != 10 );
  return 0;
}

int f1 ( int y ){  return y -= y; }

The solution.
