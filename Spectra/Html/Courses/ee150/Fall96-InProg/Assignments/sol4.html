Programming Assignment #4
Solution


Part 1:
This solution essentially involved writing an input-reading loop like that of the checkbook example we did in class, and within the loop, calling the getNumInBase with the right arguments for reading in the operands, using a switch to implement the operators, and then calling putNumInBase to write the values correctly.

Our solution is in base1.c.

Part 2:
The key thing here is to capture all of the error handling in a couple of new functions, one for reading the first operand and notifying the user about any errors in trying to do, and the other for doing a similar task with the second operand.

Our solution is base2.c.

Part 3:
The key here is to write one function to determine whether a number is prime, and another function that then calls this function for each value in a range of numbers.

Our solution is prime1.c.

Part 4:
This program extends the previous program significantly. The key is to write separate functions for printing primes in ascending and descending order. The main program then calls the appropriate function with the appropriate arguments depending on how the user answers the various questions.

Our solution is prime2.c.
