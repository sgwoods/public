Solution


Part 1:
The key to this solution is writing the main loop to read in the scores. For the purpose of this loop, we use readValues to read input just like we read input with scanf: we have a while that calls the function and compares its return value with the number of items we were trying to read.

It is also very useful to write and use a pair of functions. computeWeightedAverage takes the student's scores and the weights and returns the weighted avg. assignGrades takes the average and the minimum scores for each grade, uses a search to determine the index of the appropriate minimum score, and then uses that index into an array of letter grades to locate the specific grade to print.

Our solution is in scores.c.

Part 2:
Tackling this problem involved several changes. The first is to call functions to handle reading the number of assignments, weights, and grading scale and verifying that they are acceptable. Our solution does thi sby calling three functions obtainAssignments, obtainWeights, and obtainGrades. Only if all of these are successful do we do the rest of the main.

The second change is to add counters for the number of students and for each of the different grades. The grade counters are done as a single array and initialized to zero with assignValues. They are then updated each time we assign a grade to a student.

The last change is to update the input-reading loop so that it remember how many scores each readValues read. At the end of the loop we test this value to determine whether we processed all the scores. Only if we are successful do we print the summary info.

Our solution is in scores2.c.

Part 3:
There were several changes to make here. One is that to replace the function to obtain the number of assignments with a function that obtains both the number of assignments and the maximum numbre of students.

The next change is to use malloc to allocate the arrays that are dependent on these values. This involves changing the original array declaration to a pointer, then calling malloc to allocate the space. The rest of the program doesn't have to change.

Another change is within the input-reading loop. It now has to use insertDoubleValue to place the average it computes in the correct position in the array. And it has to exit the loop early if it runs out of array entries into which it can store the next average.

The final change is an additional function to print all of the averages and their grades. This function works by running through the sorted array of averages.

Our solution is in scores3.c.

Part 4:
There were two major changes to make here. One is to change the function for obtaining the weights to use getline to read the line if readValues claims there are no values on the line. You have to make a similar change to the function to obtain the minimum scores for the grades. The other change is to print the highest and lowest values, but that simply involves printing the array's first and last elements.
