EE150 - Spring 1996

Programming Assignment #3
Solution


Part 1:
This solution was essentially an input-reading loop, wrapped around a series of calls to the functions in timeutils.c.

Our solution is in time1.c.

Part 2:
This part involved extending the program in the previous part to compute the longest and shortest trip length, as well as the average trip length.

Our solution is time2.c.

Part 3:
The key to this solution is to write a set of general functions for doing things like writing the right endpoints or both right and left endpoints of the various numbers.

Our solution is numbers1.c.

Part 4:
This program just extends the previous program to do the remaining numbers. This primarily involves adding a function to write the left endpoints to the previous solution, and then using a combination of these general functions to write each of the numbers.

Our solution is numbers2.c.
