A method of interactive recognition of spatially defined model deployment templates using abstraction

Steven Woods.

Abstract

Abstraction as been suggested intuitively and shown experimentatlly to be an effective method of improving the efficiency of various types of problem solving applications such as scheduling and planning. In this paper we present an interactive system for model deployment template recognition that uses abstractio as the basis for both model representation and user interaction, and which provides a convenient model for iteratively improving confidence in proposed tmeplate matches or solutions.
