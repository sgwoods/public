Logging in to Wiliki

Once you see the login prompt, you need to type your login ID and then press [enter]. After that it will give you the prompt "Password:". Type your password (which won't show up on the screen). If you now get a message "Login incorrect", you'll have to type your login ID and password in once again. Once you do it correctly, you should see a prompt that looks something like "term=(vt100)". If you are using a vt100 terminal (or terminal emulation program), just hit [enter]. Be careful, though, there are also zenith terminals in Holmes 486 and hp terminals in Holmes 244. If you get the terminal type wrong, you'll see lots of garbage on the screen and will need to go to a monitor for help.

If you've done everything correctly, you should see the command prompt, usually something like "[some number] %". Congratulations. You can now get started with the real work!
